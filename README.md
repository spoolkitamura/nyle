# Nyle

'Nyle' is a minimal graphics framework using Ruby/GTK3 and rcairo.

## Requirements

Ruby/GTK3 >= 3.3.1

## Installation

Install it yourself as:

    $ gem install nyle

## Usage

```ruby
require 'nyle'
```
